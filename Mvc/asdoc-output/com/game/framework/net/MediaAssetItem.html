Classpublic class MediaAssetItem
InheritanceMediaAssetItem AssetItem UIComponent BaseSprite flash.display.Sprite

仅用于模块的加载 使用 MediaAssetItem 对象方法: 1, addMediaView 先加到显示列表中，此时就可以得到父级，同时去判断父级中是否有相同名字（name）的对象，如果有，直接返回。 2，initView 初始化 var login:MediaAssetItem = new MediaAssetItem(MediaURL.fromUIView(MediaURL.SKIN.LoginViewSkin)); uimanager.addMediaView(login); login.initView();
